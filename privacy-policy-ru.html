Политика конфиденциальности

Дата вступления в силу: 27.02.2026

Мобильное приложение STENA (далее — «Приложение», «Сервис») разрабатывается и управляется компанией DK Leading LLC.

Настоящая Политика конфиденциальности описывает, какие данные собираются, как они используются и защищаются при использовании Сервиса.

1. Оператор персональных данных

Компания: DK Leading LLC
Email для связи: admin@dkleading.com

2. Какие данные мы собираем

2.1 Регистрационные данные

  • Адрес электронной почты
  • Номер телефона (для входа через OTP)
  • Имя пользователя и профильные данные

2.2 Контакты (по разрешению пользователя)

При предоставлении доступа к контактам устройство может передать данные для поиска пользователей STENA среди ваших контактов. Контактная информация используется исключительно для сопоставления пользователей и не хранится без необходимости.

2.3 Камера и фотографии

При предоставлении разрешения Приложение может получать доступ к камере и галерее устройства исключительно для загрузки пользовательского контента по инициативе пользователя.

2.4 Пользовательский контент

  • Публикации
  • Сообщения
  • Фотографии и медиафайлы

3. Цели обработки данных

  • Аутентификация пользователей
  • Обеспечение работы социальных функций
  • Предотвращение мошенничества и злоупотреблений
  • Улучшение качества сервиса

4. Правовые основания обработки

Обработка персональных данных осуществляется на основании согласия пользователя, исполнения пользовательского соглашения, а также законных интересов по обеспечению функционирования Сервиса.

5. Передача данных третьим лицам

Мы не продаём персональные данные пользователей. Передача возможна только:

  • В рамках использования инфраструктурных сервисов
  • По требованию закона
  • Для защиты прав и безопасности пользователей

6. Безопасность данных

Передача данных осуществляется по защищённым каналам (HTTPS). Мы применяем организационные и технические меры защиты информации.

7. Срок хранения данных

Данные хранятся в течение срока действия аккаунта либо в соответствии с требованиями законодательства.

8. Права пользователя

Пользователь вправе запросить:

  • Доступ к своим данным
  • Исправление неточностей
  • Удаление аккаунта и данных
  • Ограничение обработки

Запросы направляются на: admin@dkleading.com

9. Удаление аккаунта

Пользователь может направить запрос на удаление аккаунта по электронной почте. Данные будут удалены в разумный срок, за исключением случаев, когда их хранение требуется законом.

10. Изменения политики

Мы оставляем за собой право обновлять настоящую Политику конфиденциальности. Обновления публикуются на данной странице.
